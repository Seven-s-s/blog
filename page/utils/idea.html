  • 设置作者注释

  • 设置作者注释

    准备好模板

    /**
    * @Author 张三
    * @Description
    * @Date $date$ $time$
    **/

    添加模板

    点击File → settings→Editor→live Templates

    点击“+”，先添加一个群组,点击Template Group，取名为MyTemplate

    添加一个模板名称，点击Live Template

    Abbreviation框填入“*”，Description框填入“注释模板”，Template text框填入刚才的模板

    点击Define，选择Java→Declaration

    点击Edit variables进行如下配置，点击OK

    测试

    在类名前输入*再按下Tab键，如下便成功

    /**
    * @Author 张三
    * @Description
    * @Date 2020/10/27 16:06
    **/

    在方法名前面输入@Param跟@return就会有值

  • 设置方法注释

    此操作在方法外也能获取参数

    准备好模板

    *
    $params$
    * @return
    * @Author zhangsan
    * @Date $date$
    **/

    添加模板

    Abbreviation框填入 "method"(自定义)

    Options下面的Expand with设置为Enter

    params中Expression填入以下脚本

    groovyScript("def result=''; def params=\"${_1}\".replaceAll('[\\\\[|\\\\]|\\\\s]', '').split(',').toList(); for(i = 0; i < params.size(); i++) {result+=' * @param ' + params[i] + ((i < params.size() - 1) ? '\\n' : '')}; return result", methodParameters())

    测试

    idea默认的生成注释方式为 /*+模板名+快捷键

    在方法名上输入/*method+enter

    /**
    * @param 参数
    * @return 结果
    * @Author zhangsan
    * @Date 2020/11/18
    **/
